# PxollyAPI
Библиотека для взаимодействия с API чат менеджера Pxolly (https://pxolly.ru)

## Доступные категории методов

* Database
* Users
* Utils

## Зависимости и инструменты
* Пакетный менеджер: uv
* HTTP клиент: niquests
* Валидация: pydantic
* Тестирование: pytest, pytest-asyncio
